This repository contains a Reinforcement Learning project exploring advanced exploration strategies through Intrinsic Curiosity Module (ICM) and Random Network Distillation (RND).

The objective is to improve agent exploration in sparse-reward environments by leveraging intrinsic motivation mechanisms that encourage the discovery of novel states and behaviors.

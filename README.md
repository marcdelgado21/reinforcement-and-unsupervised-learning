# Reinforcement and Unsupervised Learning Projects

This repository contains a collection of academic projects focused on
reinforcement learning and unsupervised learning, developed during my
Bachelor’s Degree in Artificial Intelligence at Universitat Politècnica
de Catalunya (UPC).

The projects include:
- Monte Carlo methods for reinforcement learning
- Q-learning and policy evaluation
- Unsupervised learning techniques and analysis

Most projects were developed collaboratively as part of university
assignments. My main contributions include algorithm implementation,
experimentation, evaluation and result analysis.
